## 👨‍🎓 Pour les Étudiants

### Étape 1 : Accéder à la section Paiements

1. Connectez-vous à votre compte étudiant
2. Dans le tableau de bord, cliquez sur l'onglet **"Paiements"** ou **"Paiements"** dans le menu de navigation

### Étape 2 : Consulter vos frais de scolarité

- Vous verrez une liste de tous vos frais de scolarité
- Chaque frais affiche :
  - **Période** (ex: "Trimestre 1", "Semestre 1")
  - **Montant** en FCFA
  - **Date d'échéance**
  - **Statut** (Payé, En attente, En retard)

### Étape 3 : Initier un paiement

1. Cliquez sur le bouton **"Payer maintenant"** à côté du frais que vous souhaitez payer
2. Un modal de paiement s'ouvrira avec les détails du frais

### Étape 4 : Choisir la méthode de paiement

Vous avez 4 options :

#### 💳 **Carte Bancaire**
- Sélectionnez l'option "Carte"
- Entrez les informations de votre carte bancaire
- Confirmez le paiement

#### 📱 **Mobile Money**
- Sélectionnez l'option "Mobile Money"
- **Choisissez votre opérateur** :
  - 🟠 Orange Money
  - 🟡 MTN Mobile Money
  - 🔵 Moov Money
  - ⚪ Autre
- **Entrez votre numéro de téléphone** (format: +237 6XX XXX XXX)
- Suivez les instructions affichées selon votre opérateur :
  
  **Pour Orange Money :**
  1. Composez `*144#` sur votre téléphone Orange
  2. Sélectionnez "Payer une facture" ou "Payer un service"
  3. Entrez le numéro de compte affiché
  4. Entrez le montant : **[Montant en FCFA]**
  5. Confirmez avec votre code PIN
  6. Entrez le code de transaction reçu par SMS dans le champ prévu

  **Pour MTN Mobile Money :**
  1. Composez `*126#` sur votre téléphone MTN
  2. Sélectionnez "Paiement de facture"
  3. Entrez le numéro de compte affiché
  4. Entrez le montant : **[Montant en FCFA]**
  5. Confirmez avec votre code PIN
  6. Entrez le code de transaction reçu par SMS

  **Pour Moov Money :**
  1. Composez `*155#` sur votre téléphone Moov
  2. Sélectionnez "Paiement"
  3. Entrez le numéro de compte affiché
  4. Entrez le montant : **[Montant en FCFA]**
  5. Confirmez avec votre code PIN
  6. Entrez le code de transaction reçu par SMS

- **Code de transaction** (optionnel) : Entrez le code reçu par SMS après avoir effectué le paiement

#### 🏦 **Virement Bancaire**
- Sélectionnez l'option "Virement"
- Entrez votre **numéro de compte bancaire**
- Entrez la **référence du virement**
- Effectuez le virement depuis votre banque
- Le paiement sera confirmé une fois le virement vérifié

#### 💵 **Espèces**
- Sélectionnez l'option "Espèces"
- Rendez-vous à l'administration de l'établissement avec le montant exact
- Le paiement sera enregistré manuellement par le personnel administratif

### Étape 5 : Enregistrer la demande de paiement

1. Vérifiez tous les détails
2. Cliquez sur le bouton de paiement ou d'enregistrement affiché dans le formulaire
3. Le paiement est créé avec le statut **"En attente"**
4. Il sera validé par l'administration, l'économe ou un webhook sécurisé du prestataire de paiement si l'intégration est active

### Étape 6 : Consulter l'historique

- Dans la section "Historique des paiements", vous pouvez :
  - Voir tous vos paiements passés
  - Télécharger les reçus (format PDF)
  - Filtrer par méthode de paiement ou statut

## 👨‍👩‍👧‍👦 Pour les Parents

1. Connectez-vous à votre compte parent
2. Sélectionnez l'enfant pour lequel vous souhaitez effectuer un paiement
3. Dans le menu de navigation, cliquez sur **"Paiements"**

### Étape 2 : Consulter les frais de scolarité

- Vous verrez tous les frais de scolarité de votre enfant
- Chaque frais affiche :
  - **Période** (ex: "Trimestre 1", "Semestre 1")
  - **Montant** en FCFA
  - **Date d'échéance**
  - **Statut** (Payé, En attente, En retard)

### Étape 3 : Initier un paiement

1. Cliquez sur le bouton **"Payer maintenant"** à côté du frais que vous souhaitez payer
2. Un modal de paiement s'ouvrira avec les détails du frais

### Étape 4 : Choisir la méthode de paiement

Les mêmes 4 options sont disponibles que pour les étudiants (voir ci-dessus).

1. Vérifiez tous les détails
2. Cliquez sur le bouton de paiement ou d'enregistrement affiché dans le formulaire

- Dans la section "Historique des paiements", vous pouvez :
  - Voir tous les paiements effectués pour votre enfant

## 💳 Méthodes de paiement disponibles

### 1. Carte Bancaire
- **Avantages** : Paiement instantané, sécurisé
- **Délai de confirmation** : Après validation sécurisée par l'administration ou par le prestataire de paiement
- **Frais** : Selon votre banque

### 2. Mobile Money
- **Avantages** : Rapide, accessible, pas besoin de compte bancaire
- **Opérateurs supportés** :
  - Orange Money
  - MTN Mobile Money
  - Moov Money
  - Autres opérateurs
- **Délai de confirmation** : Après validation sécurisée par l'administration ou par le prestataire de paiement
- **Frais** : Selon votre opérateur

### 3. Virement Bancaire
- **Avantages** : Sécurisé, adapté aux gros montants
- **Délai de confirmation** : 1-3 jours ouvrables (vérification manuelle)
- **Frais** : Selon votre banque

### 4. Espèces
- **Avantages** : Pas de frais de transaction
- **Délai de confirmation** : Immédiat après dépôt à l'administration
- **Note** : Vous devez vous rendre physiquement à l'établissement

## 🔍 Dépannage

### Le paiement ne s'affiche pas
- Vérifiez que vous êtes bien connecté
- Actualisez la page (F5)
- Vérifiez votre connexion internet

### Erreur lors du paiement Mobile Money
- Vérifiez que votre numéro de téléphone est correct (format: +237 6XX XXX XXX)
- Assurez-vous d'avoir suffisamment de crédit sur votre compte Mobile Money
- Vérifiez que vous avez sélectionné le bon opérateur
- Réessayez après quelques minutes

### Le code de transaction n'est pas accepté
- Vérifiez que vous avez bien copié le code reçu par SMS
- Le code est généralement valide pendant 24 heures
- Contactez le support si le problème persiste

### Le paiement est en "En attente"
- Pour Mobile Money : Vérifiez que vous avez bien confirmé le paiement avec votre code PIN
- Pour Virement : Le paiement sera confirmé une fois le virement vérifié (1-3 jours)
- Pour Espèces : Le paiement sera confirmé après dépôt à l'administration

### Je ne reçois pas de reçu
- Vérifiez votre dossier de téléchargements
- Cliquez à nouveau sur "Télécharger le reçu" dans l'historique
- Contactez le support si le problème persiste

### Questions ou problèmes ?
- Contactez l'administration de l'établissement
- Envoyez un email au support technique
- Consultez la section "Aide" dans l'application
